# [36_Topia] 기숙사 세탁기 사용시간 어플

## [팀 구성원 및 역할]

### 팀 구성원 : 5명
### 팀장 : 김다빈 
### 개발 구성원 : 김인덕, 김동욱 
### PPT 제작 : 김민진
### UI 디자인 : 김민정

## [프로젝트 개요] 

프로젝트명 : 36Topia 

개발 동기 : 공용 세탁기가 모두 동작 중일 때, 계속 대기해야 하는 불편함 발생 

개선점 : 공용 세탁기의 다음 사용자를 위해서 세탁 시간 종료 알림을 보내어 불편함 해소
